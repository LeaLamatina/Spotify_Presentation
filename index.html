Can you use the Audio Features from a Spotify API call and Machine Learning to predict how popular a song will be?

Objective:

Our major assumption was that a song's audio quality had bearing on how popular it would be. Based on that assumption, we used multiple different audio data points from Spotify's API calls to feed to four different machine learning alogorythms.


Data Source:

All data came from Spotify's API calls, using the SpotiPy library to increase the ease and speed of the calls. API endpoints used included Track Results, Audio Features, Track Information, and Artist Information. Documentation for both are below:

SpotiPy Library Documentation
Spotify API Documentation

Data Challanges & Limitations:

The data is based soley on the current top 10,000 streamed songs on Spotify. This means that our data is at the mercy of Spotify's key demographics and the time of the year. Data was run December 2018, meaning there are a lot of Christmas songs charting that would not any other season.

After running the API calls, it was also discovered there are quite a few duplicate songs on the list. Therefore, the original list of 10,000 gets cut down when the data is de-duped.

Q: After all of the Data Munging is done, and data is trained and tested, what are the results?

A: 33% Accuracy, or very little correlation.


Audio Features Visualizations


The data shows that the audio quality of a track has little bearing on overall popularity. There is a much stronger correlation between an artists popularity and how well a song will do.

Spotify Definitions:

free_breakfast
acousticness - float

A confidence measure from 0.0 to 1.0 of whether the track is acoustic. 1.0 represents high confidence the track is acoustic.

headset
danceability - float

Danceability describes how suitable a track is for dancing based on a combination of musical elements including tempo, rhythm stability, beat strength, and overall regularity. A value of 0.0 is least danceable and 1.0 is most danceable.

plus_one
energy - float

Energy is a measure from 0.0 to 1.0 and represents a perceptual measure of intensity and activity. Typically, energetic tracks feel fast, loud, and noisy. For example, death metal has high energy, while a Bach prelude scores low on the scale. Perceptual features contributing to this attribute include dynamic range, perceived loudness, timbre, onset rate, and general entropy.

volume_up
loudness - float

The overall loudness of a track in decibels (dB). Loudness values are averaged across the entire track and are useful for comparing relative loudness of tracks. Loudness is the quality of a sound that is the primary psychological correlate of physical strength (amplitude). Values typical range between -60 and 0 db.
