API Calls


SpotiPy Library does most of the heavy lifing. But here is how we got all the data needed:

  1. Calls went out to get the 10,000 most popular songs, and everything is thrown into a Pandas DataFrame.
  2. Duplicated titles were dropped for data integrity.
  3. Track ID's were used to send out another call to get the Audio Features. These were merged into the first DataFrame.
  4. Before proceeding, go down a rabbit hole and look at the data because Spotify returns so much amazing stuff. Regroup, get back on task.
  5. Use Track ID one more time to send out a call and return the Artist ID. This is nowhere in the previous data which is very frustrating.
  6. After getting the Artist ID, send out one final call to return Genre, Artist Popularity, and Artist Image.

In all, it takes four API calls to collect everything
